# FileTracking

Задача: Создание программы для отслеживания состояния файлов.

Изменение состояний:
  - 1.файл не существует => файл существует
  - 2.Файл существует => файл не существует
  - 3.файл изменил размер
  
Реализация: через сигналы и слоты QT
